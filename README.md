# DOALI
Este repositório é destinado ao projeto Doali, desenvolvido por intermédio da Discplina de Programação Web, ministrada pelo Professor José Roberto do curso Técnico em Informática do IFCE. 

♾ **O Doali surge como uma plataforma de orientação para pessoas interessadas em realizar doações. Ao oferecer instruções relacionadas ao processo de doação e ao conectar indiretamente doadores a instituições de caridade, esta plataforma atua como um fator intermediário no desenvolvimento do diálogo entre esses dois agentes.**

💻 **Desenvolvimento:**
 - Uso do Node JS para desenvolvimento web;
 - Template Engine EJS (Embedded JavaScript Templating).

🛠 **IDE utilizada: Replit**

🏗 **O projeto está na primeira etapa de desenvolvimento e sujeito a modificações.**
